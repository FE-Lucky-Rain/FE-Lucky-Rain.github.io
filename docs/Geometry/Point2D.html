Cheat sheet

Name Description Options
Point2D(‘Name’) Creat Line2D object
AddPoint(obj,x,y) Add points to Point2D ‘polar’, ‘delta’
Plot(obj,varargin) Plot points ‘plable’ , ‘grid’ , ‘group’,’equal’
GetPoint(obj,id,t) Get point data
GetNpts(obj) Get number of points
GetNgpts(obj) Get number of points group
CompressNpts(obj,varargin) Compress repeated points ‘all’
Dist(x,y,obj,varargin) Calculate the closet distance to serval points ‘group’

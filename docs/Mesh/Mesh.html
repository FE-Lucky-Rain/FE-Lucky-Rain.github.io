TetGen

I use TetGen to generate tetrahedralization mesh.s

For a set of 3d (weighted) points, TetGen generates the Delaunay and weighted Delaunay tetrahedralization as well as their duals, the Voronoi diagram and power diagram. The flowchart of the mesh generation process of TetGen is shown below.

Mesh quality

The quality tetrahedral meshes of a PLC generated by the commands: -pq2/0, -pq1.4/0, and -pq1.1/0

MeshSphere(obj,n,r,varargin) Create Sphere Mesh
